brand name usage

brady UX: always written in lowercase, with "UX" in uppercase.

hélo: always lowercase with the accent on the "e".

headers and titles

all headers and titles should start with lowercase letters. key terms within headers may be capitalized to highlight importance but only if it follows the pattern: "brady UX" or "hélo".

example:

  • what we do
  • human experience, leadership and optimization

body text

sentences start with lowercase letters unless they include proper nouns or specific brand names like "brady UX" or "hélo". proper nouns within sentences are capitalized normally.

example:

  • "hi, I'm brady starr, co-founder and head of product."
  • "our mission is to improve people's time with delightful and intuitive design."

bullet points

start each bullet point with a lowercase letter. if a bullet point contains multiple sentences, follow the body text capitalization rules.

example:

  • what we do
    • product design: creating intuitive and visually engaging product designs.
    • user research: conducting in-depth user research to gather insights.

section names

use lowercase for all section names. follow the same rules for headers and titles for any subsections.

example:

  • mission
  • core values

email signatures

email signatures should follow the same capitalization rules.

example:

olya androsik
co-founder, head of global UX design operations
olya@bradyux.com

brady starr
co-founder, head of product
brady@bradyux.com

quotes and callouts

quotes and callouts should follow the body text rules.

example:

  • "we create stunning, user-centric digital experiences that blend traditional art skills with modern digital techniques."
